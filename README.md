# Transportation Analytics Dashboard

## Introduction

This project presents an interactive **Power BI dashboard** built from Uber trip data. The dashboard provides visual insights into ride patterns, trip durations, distances, and more, helping to analyze urban mobility and transportation trends.

## Installation

To run the Power BI dashboard locally:

1. Install **Microsoft Power BI Desktop** from the [official site](https://powerbi.microsoft.com/desktop/).
2. Clone this repository or download the files.
3. Open the `Transportation Analytics.pbix` file using Power BI.

## Usage

1. Load the data from `Uber Trip Details.xlsx` (should load automatically if in the same directory).
2. Explore the dashboard for visual analytics on Uber trip data.
3. Filter and interact with the visuals to gain insights into ride frequency, time distribution, and more.

## Features

- Interactive Power BI visuals and slicers
- Time-based trip analysis
- Pickup vs drop-off comparison
- Distance and duration summaries
- Heatmaps and charts for trend exploration

## Data Source

- **File:** `Uber Trip Details.xlsx`
- **Fields may include:** Date/Time, Pickup/Dropoff Location, Trip Duration, Distance, and Cost.

> Note: Ensure the Excel file remains in the same folder as the `.pbix` file for seamless data loading.

## Dependencies

- [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
- Excel-compatible environment (for data updates)

## Configuration

No additional configuration is required. Make sure the Excel file path remains consistent for data refresh.

## Troubleshooting

- **Data not loading?**
  - Check that the Excel file path is correctly linked.
  - Refresh the data source from within Power BI (`Home > Refresh`).
- **Missing visuals?**
  - Reopen the `.pbix` file and ensure data is correctly loaded.
